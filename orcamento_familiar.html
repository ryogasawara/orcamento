☁️ Sincronizar com Google Drive

Seus dados ficam num arquivo orcamento_familiar.json no seu Google Drive. Qualquer dispositivo que abrir este app e fizer login verá os mesmos dados em tempo real.

📋 Passo a passo para obter o Client ID:
1. Acesse console.cloud.google.com
2. Crie um projeto (ex: "Orçamento Familiar")
3. Menu → APIs e Serviços → Biblioteca
4. Busque e ative "Google Drive API"
5. Menu → APIs e Serviços → Credenciais
6. Clique "Criar credenciais → ID do cliente OAuth"
7. Tipo: Aplicativo da Web
8. Em "Origens JavaScript autorizadas" adicione: file://
9. Copie o ID do cliente e cole abaixo
